# SweetyCodes Portfolio

Welcome to my personal portfolio! This site is designed to showcase my journey as a Full-Stack Developer and AI Engineer. It serves as both a project showcase and an interactive resume.

## Key Features

**AI Chatbot Assistant:** An intelligent assistant trained on my professional background. It supports free-movement on the screen and provides context-aware answers to inquiries about my skills and projects.

 **Dynamic Aesthetic:** Weather-app features dynamic video backgrounds and audio that adapt to the time of day, creating a lively user experience.

 **Bilingual Support:** Full English/Korean language toggling for global accessibility.

 **Project Gallery:** Interactive project showcase with lightbox zoom functionality and detailed case studies.

## Tech Stack

 **Framework:** Next.js 15, React

**Styling:** Tailwind CSS, Framer Motion (for animations)

**AI/LLM:** Gemini API (Chatbot), OpenAI API (Automation projects) 16 *
 **Deployment: ** Vercel


## Projects Overview


**AI Chatbot Desktop Automation:** An AI-powered tool for desktop interaction and intelligent message replying.

**Vibe Weather (EN/KO):** A high-performance, bilingual weather app with real-feel calculations and dynamic assets.

 **Sales Analysis Reporter:** Python-based data pipeline for professional business reporting.

 **Batch File Organizer:** Efficient file system automation tool with audit logging.

## Getting Started

 1. **Clone the repository:**

git clone https://github.com/sweety-HJH223/portfolio.git

 2. **Install dependencies:**

npm install

 3. **Configure environment variables:**

 Create a .env.local file in the root directory and add your API key:

GEMINI_API_KEY=your_actual_api_key_here

 4. **Run the development server:**

npm run dev
